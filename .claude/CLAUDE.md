## How to Use

When creating a new project:
1. Run the appropriate installer script (`claude_rules.sh` for Claude Code or `cursor_rules.sh` for Cursor IDE)
2. The script will download and set up the rules in your project
3. Start coding with AI assistance that follows these rules
